# Portfolio Windows 7 - HERMOCRATE

Portfolio interactif en HTML, CSS et JavaScript, pensé comme un faux système Windows 7.

## Contenu

- Bureau avec icônes interactives
- Fenêtres déplaçables
- Explorateur de projets
- Fiches projets : Yo-Ji-Hu, Calories Tracker Pro, Audit SQL, Mandat Bala
- Captures de projets agrandissables
- Internet Explorer avec page GitHub, jeu Windows Defender directement sur le bureau et FAQ recruteur
- Formulaire de contact via FormSubmit
- Easter egg Matrix

## Lancer en local

Avec Laragon, place le dossier dans :

```text
C:\laragon\www\portfolio-windows7
```

Puis ouvre :

```text
http://portfolio-windows7.test
```

Ou lance simplement avec un serveur local.
